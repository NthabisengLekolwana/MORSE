# Morse Code Converter  

About  
This is a simple and easy to use Morse Code Converter built using HTML, CSS, and JavaScript. It allows users to convert text to Morse code and vice versa, making it a fun and educational tool for exploring Morse code communication.  

 Features  
1 Convert text to Morse code 
2 Convert Morse code to text 
3 User-friendly interface with a blue-themed design  
4 Real-time conversion without page reload    

 Technologies Used  
1 HTML: Structure of the application  
2 CSS: Styling with shades of blue for a smooth UI  
3 JavaScript: Logic for Morse code conversion  

 How to Use  
1 Open the webpage.  
2 Type text into the input box and get Morse code instantly.  
3 Type Morse code using dots (`.`) and dashes (`-`) and get the original text instantly.
